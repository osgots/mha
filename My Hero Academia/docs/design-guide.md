# My Hero Academia Universe — Design Guide

## Design Philosophy

The My Hero Academia Universe website combines immersive anime aesthetics with modern, professional web design. It is crafted to feel energetic, interactive, and visually compelling while maintaining excellent usability, accessibility, and performance.

## Visual Identity

* **Primary Style:** Anime-inspired futuristic design
* **Audience:** Anime fans, manga readers, collectors, and Gen Z users
* **Tone:** Bold, energetic, heroic, and modern
* **User Experience:** Smooth, intuitive, and immersive

## Color Palette

* Primary: Hero Blue (`#0EA5E9`)
* Secondary: Plus Ultra Yellow (`#FACC15`)
* Accent: Crimson Red (`#EF4444`)
* Dark Background: Midnight Black (`#0F172A`)
* Surface: Slate Gray (`#1E293B`)
* Text Primary: White (`#F8FAFC`)
* Text Secondary: Light Gray (`#CBD5E1`)

## Typography

* Primary Font: Poppins
* Secondary Font: Inter
* Accent Font: Bebas Neue

## Layout Principles

* Mobile-first responsive design
* Grid-based content organization
* Generous spacing for readability
* Clear visual hierarchy
* Consistent component styling

## UI Components

* Sticky navigation bar
* Hero image slider
* Character cards with hover effects
* Search modal
* Theme switcher
* Audio player controls
* Buy Me a Coffee support section

## Animation Guidelines

* Smooth transitions (0.3s–0.5s)
* Subtle hover scaling
* Fade-in scroll reveals
* Hero slider motion effects
* Parallax background enhancements

## Accessibility Standards

* Semantic HTML5 structure
* Keyboard navigation support
* Proper ARIA labels
* High contrast ratios
* Alt text for all images

## Performance Goals

* Lazy-loaded images
* Optimized assets
* Minified CSS and JavaScript for production
* Efficient DOM manipulation
* Fast initial page load
